The script will collect the current OS information, browser metadata (including **wallet data**) and upload it.

What's more serious is that it will scan all files under `Desktop`, `Documents`, `Downloads` and compress them for upload.

If the mnemonic file is contained in these directories, it will be stolen immediately by scammers.

## What's Next

Unfortunately, the only information we can get from the scammers is these server ports. We can only take some retaliatory measures against these ports.

After executing several times, the server will no longer be able to respond normally.
